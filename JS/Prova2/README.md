# Instalar
Precisa ter o Node Instalado no PC, sugiro usar o NVM

## Instalar dependencias
Rode o seguinte comando:
- `npm i`

# Buildar o Código
`npm run build`

# Rodar o código
`npm run start`

# Buildar e rodar direto
`npm run test`
